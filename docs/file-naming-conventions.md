# Component Files

- **Components**: `PascalCase.tsx` (e.g., `UserCard.tsx`, `ProjectStatusBadge.tsx`)
- **Component name must match filename**

# Test Files

- **Tests**: `ComponentName.test.tsx` or `utilFunction.test.ts`
- **Co-locate tests** with the files they test

# Story Files

- **Storybook**: `ComponentName.stories.tsx`

# Support Files

- **Types**: `ComponentName.types.ts` or `domain.types.ts`
- **Constants**: `ComponentName.constants.ts` or `domain.constants.ts`
- **Hooks**: `useHookName.tsx` (hook name must match filename)
